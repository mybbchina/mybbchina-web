先进, 直观

MyBB的管理员控制面板拥有最先进的控制器

MyBB的管理员控制面板非常先进并且允许广泛的自定义, 而且它的布局非常直观并且易于使用. 例如, 论坛的权限系统控制着非常大量的设置, 如何使这些设置能够分门别类并且在你要使用它的时候很快就能找到, 包括从组和版块继承权限, 迅速且深入的权限编辑器, 还有从别的版块复制权限到另一个版块. MyBB的管理员控制面板被设计为能够非常方便的执行大多数的基本改变和全面的个性化, 这也是大多数论坛管理员需要的.

制作主题是轻而易举的事情

个性化你的论坛外观非常简单

MyBB的主题制作系统允许您直接在管理员控制面板中通过几次点击就可以个性化您的论坛. 不光是颜色和一般的样式, 还可以深入到模板和HTML. 你可以使一个现有的MyBB主题变得完全不同. 另外, 主题管理器使用AJAX技术加速载入速度并且可以单独保存同一个CSS文件中不同的CSS项目. 您不必钻研PHP和巨大的样式表; 您只需要一点 CSS 和 HTML 的基本知识. 您也可以通过高级CSS编辑器直接编辑CSS文件, 这适用于高级用户.
