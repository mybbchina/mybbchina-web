管理员控制面板

从底层代码重新开发

MyBB 1.4包含一个全新的管理员控制面板，已经从底层代码完全重新开发过的管理员控制面板, 更加容易使用，同时让您完全控制您的论坛。在论坛业丰富的经验告诉我们一件事：控制面板设计不当导致越来越多的失望和论坛维护难度的增加。我们已经花了时间规划MyBB控制面板，并重新开发，以确保它易于使用，并且提高效率。

整合到六个逻辑区域

通常大家登陆后台就是为了做一下这些相关的事情: 配置, 用户和组的管理或者版块和内容的管理. 我们已经打破了以前的控制面板格局以便于你能一眼就看到你需要的, 当你需要以下某个功能的时候:

  • 主页 - 您论坛的概述 (控制台)
  • 配置 - 配置论坛相关的所有选项都在这里 (设置, 封禁, 日历等)
  • 版块和帖子 - 内容管理 (版块, 公告, 权限等)
  • 用户和组 - 用户和组的管理 (用户, 组, 晋升, 邮件群发等)
  • 模板和风格 - 设计管理 (模板, 主题等)
  • 工具和维护 - 一般系统实用工具 (系统健康, 日志, 备份等)

强大的可管理的用户查看器

用户搜索结果可以保存为用户查看器. 例如, 你可能想创建一个用户查看器, 其中的用户主要是论坛工作人员, 比如管理员, 超级版主和版主. 这个搜索结果就可以保存成一个用户查看器, 你甚至可以给它起个名字叫做"论坛工作人员", 日后想快速查看这些用户只需点击一下就可以完成, 而不必每次都设定条件去查询. 当然, 你也可以设定只显示你需要的信息, 比如用户名, 电子邮件地址, 注册时间等.
