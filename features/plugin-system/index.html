插件系统

MyBB的插件系统是当今论坛软件中最先进的插件系统之一. 它可以在几分钟内安装好一个插件, 并且使您的论坛拥有您想要的功能. 热情而友好的MyBB社区已经开发了大量的插件供大家使用; 其中有的插件是用来集中显示未读主题或者由新回复的主题或者没有处理的主题, 有的是用来扩展论坛的统计系统,让统计资料更详细更丰富, 有的是用来显示不同版块不同用户的权限, 有的是用来进行搜索引擎优化, 还有其它狠多很多各种各样的插件.

管理员控制面板插件管理器

钩子无处不在

插件的钩子贯穿于整个MyBB

MyBB的插件系统利用钩子可以运行于整个PHP代码. 这些钩子允许插件在放置钩子的地方运行代码, 这意味着大多数插件不需要修改任何文件. 钩子也可以在 管理员控制面板 中找到, 意思是说插件可以在管理员控制面添加一个新页面或者在一个已经存在页面中添加一个项目, 同样也不需要编辑某个文件. MyBB的插件系统应用非常广泛, 这意味着您完全不用修改文件就可以为MyBB添加一个新的功能, 无论这个功能是大是小.

管理员控制面板中的模板编辑

新模板? 没问题!

插件也可以创建或者修改模板

MyBB的插件系统不光是允许通过钩子来运行代码. 当你激活一个插件, 他们可以添加一个新的模板, 也可以修改现有模板. 因为插件也可以修改你的模板, 而且不需要冗余的步骤, 你只需要上传一个插件文件, 点击激活, 这样就好了. 个性化您的论坛再也没有比这更简单的了!
