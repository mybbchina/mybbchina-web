Configurable Search Engine Detection

Add detection rules for search engines

MyBB 1.1 began recognizing search engines that were crawling your forum. MyBB 1.4 extends upon this functionality by allowing full management of detected search engines via the control panel. Administrators can easily add new detection rules for search engines and customize the existing ones.

Change what search engines see

It's also possible to configure which theme each spider sees, the language they'll browse the forums with and which user group they should be placed in.

Know when you were last crawled

As MyBB automatically detects search engines as they crawl your forums, it's possible for it to let you know when they last visited too. The MyBB 1.4 Control Panel when listing the detectable search engines will also let you know the date & time they last visited.
