MyBB 1.4 功能浏览

MyBB 1.4 更新内容中的亮点

  • 管理员控制面板
    完全重新开发, 让您可以完全控制您的论坛.
  • 版主控制面板
    供论坛版主和工作人员使用的功能强大的工具.
  • 内置 模板 & 主题 编辑器
    内置语法高亮的模板编辑器和简单的CSS编辑器, 可以个性化外观.
  • 基于AJAX的快速恢复
    使用AJAX技术快速回复一个主题, 使您的回复立刻出现在页面中.
  • DBMS 支持
    使用内置的 replication/slave 系统可以支持 MySQL, PostgreSQL 和 SQLite 数据库.
  • 日历和事件
    多个日历每一个都可以有自己的循环事件，或由用户自行排列的事件。
  • 警告系统
    当用户行为不当时自动发出警告或惩罚.
  • 用户晋升
    基于用户发帖数, 声望或者注册日期的自动晋升系统.
  • 可配置的搜索引擎检测
    当一个搜索引擎抓取您的论坛时可以被准确的记录并且可以配置他们能抓取那些内容。
  • 搜索引擎友好的链接
    使用搜索引擎友好的URL来增加你在各大搜索引擎的收录.
  • 群发邮件系统
    发送即时或预定的电子邮件和短消息给匹配指定的标准的注册用户。
  • 强大的插件系统
    改进了的插件系统可以更大的扩展你的论坛，而不需要修改代码。
  • 计划任务系统
    在后台自动运行您设置的任务。
  • 增强的用户界面
    再没有比现在更容易的了, 这全部要归功于页面的重新设计和强大的AJAX应用.
  • 性能提升
    MyBB 1.4 比之前的版本运行速度更快, 效率更高.
