MyBB 1.4 Feature Tour

MyBB 1.4 is a major release of the most powerful, free PHP and MySQL forum software available. There's over 50 new features, many behind the scenes optimizations and a complete rewrite of the Administrator Control Panel. Take a look at the tour below for a small snapshot of exactly what's new and improved in MyBB 1.4 or take a look at the full list.

Improved User Interface

We're always working to make MyBB easier to use

MyBB 1.4 included a wide range of interface improvements that did not necessarily add any new features, but updated several outdated sections and made other features significantly easier to use.

Brand new post editor & toolbar

Included in 1.4 is a brand new post editor and toolbar that is extremely customizable and even easier to use. In fact, to emphasize how customizable it truly is, we've included an Office 2007 theme, in addition to the default MyBB theme.

Refined user control panel

The User Control Panel has seen some slight changes to make it friendlier and more welcoming. The menu has been reorganized and cleaned up, featuring three categories (Messanger, Your Profile, and Miscellaneous) that are collapsible. Also, each link in the menu now has an icon to help distinguish it from other pages.

The Edit Buddy and Ignore List page has also seen a complete overhaul. It now features AJAX to add and remove buddies and ignored users, and autocomplete when adding users. You can also now see each person's status right from this page.

New, optional horizontal post layout

MyBB 1.4 offers a new and optional horizontal post layout. Not only can administrators set the default post layout for guests, but users can also choose whether they would like the new, horizontal post layout or the classic post layout from the User CP.

Other user interface improvements

  • Revamped member listing
  • Ability to select buddies as recipients when composing Private Messages
  • Updated thread display, including new AJAX thread rating
